1: C:\Program Files\Java\jdk-9.0.1\

2: C:\Program Files\Java\jdk-9.0.1\bin

3:  javac "nomedoarquivo" [opções]
	Ex.: javac HelloWorld.java
	
4:  java "nomedoarquivo"
	Ex.: java HelloWorld
	
5:  São variaveis globais do sistema. acessaveis por todos os programas do Sistema Operacional. Servem para
	armazenar diretorios de executaveis, como PATH que armazena executaveis do java e javac.
	
6:  Apenas o administrador do sistema

7:  a. Em Pesquisar, procure e selecione: Sistema (Painel de Controle)
	b. Clique no link Configurações avançadas do sistema.
	c. Clique em Variáveis de Ambiente. Na seção Variáveis do Sistema, localize a variável de ambiente PATH 
	e selecione-a. Clique em Editar. Se a variável de ambiente PATH não existir, clique em Novo.
	d. Na janela Editar Variável de Sistema (ou Nova Variável de Sistema), especifique o valor da variável
	de ambiente PATH. Clique em OK. Feche todas as janelas restantes clicando em OK.
	e. Reabra a janela Prompt de comando e execute o código Java.
	
8.  JAVA_HOME é uma convenção para ser utilizada como variavel de sistema para outros programas encontrarem
	o Java no computador.
